  • Qu'est-ce qu'un language de programation ?

    Informations tirés de wikipedia.org

    "Un langage de programmation est une notation conventionnelle destinée à formuler des algorithmes et produire des programmes informatiques qui les appliquent. D'une manière similaire à une langue naturelle, un langage de programmation est composé d'un alphabet, d'un vocabulaire, de règles de grammaire et de significations."

    Pour simplifier le texte cité, un language de programmation est un genre de "document" avec une notation spéciale qui peut être compris par votre ordinateur. Il existe différents languages adapté plus ou moins à certaines utilisations comme par exemple le php pour le web ou le java pour l'embarqué.

  • Avant tout, un peu de contexte !

    Informations tirés de freejavaguide.com

    • Un language simplifié juin 1991

      Java a été lancé comme un projet appelé «Oak» par James Gosling. Les objectifs de Gosling étaient d'implémenter une machine virtuelle et un langage qui avaient une notation familière du language C mais avec une plus grande uniformité et simplicité que celui-ci.

    • Publication 1995

      La première implémentation publique a été Java 1.0. Elle a fait la promesse de "Write Once, Run Anywhere" ("écrivez une fois, utilisez partout" En français X)), avec des runtimes gratuits sur les plates-formes populaires.

    • Rachat 1997

      Sun a contacté l'organisme de normalisation ISO / CEI JTC1 et plus tard Ecma International pour officialiser Java, mais il s'est rapidement retiré du processus. Java reste un standard propriétaire de facto contrôlé par le Java Community Process.
